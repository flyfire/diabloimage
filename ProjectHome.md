# 介绍 #
使用GAE数据库存储图片。
最新的sa3upload网盘
http://sa3.org/program/sa3upload/

大菠萝相册的升级版本 http://sa3.org/program/gae-album/

相册交流QQ群：86318295

# 特点 #

  * 随意外链
  * 流量无限制(10.00 Gbytes/d)
  * 绑定自己域名
  * 图片不压缩，无水印

# 0.02更新 #

# 安装 #

  1. 修改src/app.yaml，将<application id>替换为您的App ID.
  1. 使用“appcfg.py update 目录地址”命令，将代码上传到App.
